# Gestión de Citas

## Ver Calendario de Citas

1. Desde el menú principal, toca **"Calendario"**
2. Verás tu calendario mensual con todas las citas
3. Las citas se muestran por día y hora
4. Usa las flechas para navegar entre meses

## Ver Lista de Citas

1. Desde el menú principal, toca **"Citas"**
2. Verás la lista de todas tus citas
3. Filtra por:
   - Estado (pendiente, confirmada, completada, cancelada)
   - Fecha
   - Paciente
   - Servicio

## Crear Nueva Cita

1. Toca el botón **"+"** o **"Nueva Cita"**
2. Completa el formulario:
   - **Paciente:** Selecciona de tu lista o busca
   - **Fecha:** Selecciona la fecha
   - **Hora:** Selecciona la hora disponible
   - **Servicio:** Selecciona el servicio médico
   - **Motivo:** Describe el motivo de la cita
   - **Duración:** Tiempo estimado (por defecto según servicio)
3. Toca **"Guardar"**
4. El paciente recibirá una notificación

## Editar Cita

1. En la lista o calendario, toca la cita
2. Toca **"Editar"**
3. Modifica la información necesaria:
   - Fecha/hora
   - Servicio
   - Motivo
4. Toca **"Guardar"**
5. El paciente recibirá una notificación de cambio

## Cancelar Cita

1. En los detalles de la cita, toca **"Cancelar"**
2. Confirma la cancelación
3. Opcionalmente, agrega un motivo
4. El paciente recibirá una notificación

## Crear Cita Instantánea (Zoom)

Si tienes Zoom configurado:

1. Toca **"Cita Instantánea"**
2. Selecciona el paciente
3. Se creará una reunión Zoom inmediata
4. Comparte el enlace con el paciente

## Configurar Horarios de Atención

1. Ve a **Configuración** → **Horarios**
2. Para cada día de la semana:
   - Activa/desactiva el día
   - Configura horario de inicio y fin
   - Configura tiempo de consulta (slots)
   - Configura descansos (opcional)
3. Toca **"Guardar"**

## Ver Detalles de Cita

Al tocar una cita verás:
- Información del paciente
- Fecha y hora
- Servicio
- Motivo
- Estado
- Enlace de Zoom (si aplica)
- Historial de cambios

## Completar Cita

1. Después de la consulta, ve a los detalles de la cita
2. Toca **"Completar Cita"**
3. Opcionalmente, crea la consulta médica desde aquí
4. La cita cambiará a estado "Completada"

## Recordatorios Automáticos

Las citas tienen recordatorios automáticos:
- **24 horas antes:** Notificación push y email
- **2 horas antes:** Notificación push
- El paciente también recibe recordatorios
