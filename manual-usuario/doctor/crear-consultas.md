# Crear Consultas Médicas

## Iniciar Nueva Consulta

1. Ve a los **detalles del paciente**
2. Toca **"Nueva Consulta"** o **"Iniciar Consulta"**
3. Se abrirá el formulario de consulta

## Completar Información de Consulta

### Información Básica

- **Fecha de Consulta:** Se establece automáticamente (puedes cambiarla)
- **Motivo de Consulta:** Describe el motivo principal
- **Tipo de Consulta:** Primera vez, Seguimiento, Urgencia

### Antecedentes

1. Revisa los antecedentes del paciente
2. Agrega o modifica:
   - **Antecedentes Personales Patológicos**
   - **Antecedentes Personales No Patológicos**
   - **Antecedentes Familiares**
   - **Alergias**

### Signos Vitales

Completa los signos vitales:
- **Presión Arterial:** Sistólica/Diastólica
- **Frecuencia Cardíaca:** Latidos por minuto
- **Frecuencia Respiratoria:** Respiraciones por minuto
- **Temperatura:** Grados Celsius
- **Peso:** Kilogramos
- **Talla:** Centímetros
- **IMC:** Se calcula automáticamente

### Exploración Física

Describe la exploración física:
- **Cabeza y Cuello**
- **Tórax**
- **Abdomen**
- **Extremidades**
- **Neurológico**
- **Otros hallazgos**

### Diagnóstico

1. Toca **"Agregar Diagnóstico"**
2. Busca el código CIE-10:
   - Usa el buscador de CIE-10
   - O ingresa el código directamente
3. Agrega descripción del diagnóstico
4. Puedes agregar múltiples diagnósticos

### Notas Médicas

1. En **"Notas Médicas"**, describe:
   - Evolución del paciente
   - Plan de tratamiento
   - Indicaciones
   - Próxima cita recomendada

### Consentimiento Informado

Si aplica:
1. Selecciona el tipo de consentimiento
2. El paciente debe aceptar (si está presente)
3. Se registra fecha y hora de aceptación

## Guardar Consulta

1. Revisa toda la información
2. Toca **"Guardar Consulta"**
3. La consulta se guardará en el historial del paciente
4. Se generará automáticamente un registro de auditoría

## Generar PDF de Consulta

1. Después de guardar, o desde los detalles de consulta
2. Toca **"Generar PDF"**
3. El PDF incluirá:
   - Información del paciente
   - Fecha de consulta
   - Signos vitales
   - Exploración física
   - Diagnóstico
   - Notas médicas
4. Puedes compartir o descargar el PDF

## Generar Receta Médica

1. Desde la consulta, toca **"Generar Receta"**
2. Agrega medicamentos:
   - Nombre del medicamento
   - Dosis
   - Frecuencia
   - Duración del tratamiento
3. Agrega indicaciones adicionales
4. Toca **"Generar PDF"**
5. La receta se genera en formato PDF

## Ver Consultas Anteriores

1. En los detalles del paciente, ve a **"Consultas"**
2. Verás todas las consultas ordenadas por fecha
3. Toca una consulta para ver detalles completos
4. Puedes editar consultas recientes (si tienes permisos)
