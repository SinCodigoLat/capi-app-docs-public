# Primeros Pasos

Guía rápida para comenzar a usar CAPI App.

## Registro de Cuenta

### Para Doctores

1. Abre la aplicación CAPI
2. Selecciona "Crear Cuenta"
3. Selecciona "Soy Doctor"
4. Completa el formulario:
   - Email
   - Contraseña
   - Información profesional
5. Acepta los términos y condiciones
6. Verifica tu email
7. Completa tu perfil profesional

### Para Pacientes

1. Abre la aplicación CAPI
2. Selecciona "Crear Cuenta"
3. Selecciona "Soy Paciente"
4. Completa el formulario básico
5. Verifica tu email
6. Completa tu perfil de paciente

## Inicio de Sesión

1. Abre la aplicación
2. Ingresa tu email y contraseña
3. Si olvidaste tu contraseña, usa "Olvidé mi contraseña"
4. Una vez autenticado, serás redirigido según tu rol

## Configuración Inicial

### Doctores

1. **Completa tu Perfil Profesional:**
   - Especialidad
   - Cédula profesional
   - Descripción
   - Horarios de atención

2. **Configura tu Clínica:**
   - Nombre de la clínica
   - Dirección
   - Teléfono
   - Logo (opcional)

3. **Configura Horarios:**
   - Días de atención
   - Horarios por día
   - Tiempo de consulta

4. **Configura Pagos (Opcional):**
   - Stripe Connect para recibir pagos
   - Configuración de servicios de pago

5. **Configura Zoom (Opcional):**
   - API Key de Zoom
   - Enlace instantáneo

6. **Selecciona tu Plan:**
   - Starter, Premium o Elite
   - Completa el pago de suscripción

### Pacientes

1. **Completa tu Perfil:**
   - Información personal
   - Datos médicos básicos
   - Contacto de emergencia

2. **Busca tu Doctor:**
   - Usa el buscador de doctores
   - Filtra por especialidad, ciudad
   - Revisa perfiles y disponibilidad

## Navegación Principal

### Menú de Doctor

- **Estadísticas** - Dashboard con métricas
- **Calendario** - Vista de citas
- **Conversaciones** - Chats con pacientes
- **Pacientes** - Lista de pacientes
- **Perfil** - Tu perfil profesional
- **Mi Plan** - Gestión de suscripción

### Menú de Paciente

- **Mis Citas** - Citas agendadas
- **Mis Doctores** - Doctores que te atienden
- **Conversaciones** - Chats con doctores
- **Mi Perfil** - Tu perfil de paciente

## Notificaciones

### Habilitar Notificaciones

1. La app solicitará permisos al iniciar
2. Acepta las notificaciones para recibir:
   - Recordatorios de citas
   - Mensajes de chat
   - Actualizaciones importantes

### Configurar Preferencias

1. Ve a Perfil → Notificaciones
2. Configura qué notificaciones recibir
3. Configura horarios de silencio (opcional)
