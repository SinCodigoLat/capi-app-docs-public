# Chat con Doctores

## Iniciar Conversación

### Desde Mis Doctores

1. Ve a **"Mis Doctores"**
2. Toca un doctor
3. Toca **"Iniciar Chat"**
4. Se abrirá la conversación

### Desde una Cita

1. En los detalles de una cita, toca **"Chat"**
2. Se abrirá la conversación con ese doctor

## Ver Conversaciones

1. Desde el menú principal, toca **"Conversaciones"**
2. Verás todas tus conversaciones con doctores
3. Las conversaciones se ordenan por última actividad
4. Verás:
   - Foto del doctor
   - Nombre
   - Último mensaje
   - Hora del último mensaje
   - Contador de mensajes no leídos

## Enviar Mensajes

1. Abre una conversación
2. Escribe tu mensaje en el campo de texto
3. Opcionalmente:
   - Adjunta una imagen
   - Adjunta un archivo
4. Toca el botón de enviar

## Recibir Mensajes

Cuando recibas un mensaje:
- Verás una notificación push
- El mensaje aparecerá en la conversación
- El contador de no leídos se actualizará

## Leer Mensajes

1. Abre la conversación
2. Los mensajes se marcan como leídos automáticamente
3. Verás quién envió cada mensaje y la hora

## Notificaciones de Chat

### Configurar Notificaciones

1. Ve a **Perfil** → **Notificaciones**
2. Configura:
   - Notificaciones push de chat
   - Notificaciones de escritorio (web)
   - Sonidos de notificación
   - Horarios de silencio

### Recibir Notificaciones

Recibirás notificaciones cuando:
- Recibas un mensaje nuevo
- Un doctor te mencione
- Haya actividad importante en el chat

## Archivos en Chat

### Enviar Archivo

1. En el chat, toca el ícono de adjuntar
2. Selecciona:
   - **Imagen:** Desde galería o cámara
   - **Archivo:** Documento PDF u otro
3. El archivo se enviará y el doctor podrá verlo

### Ver Archivos Enviados

1. En la conversación, los archivos aparecen como mensajes
2. Toca un archivo para:
   - Ver imagen (ampliada)
   - Descargar documento
   - Compartir

## Historial de Chat

Todas las conversaciones se guardan:
- Puedes ver mensajes anteriores
- El historial se mantiene mientras la conversación exista
- Puedes buscar en el historial
